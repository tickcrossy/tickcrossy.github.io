摘要：有一些是翻 blog 的时候发现的，还有一些是刷 ProjectEuler 的时候自己随便口糊的。

设 A,B∈Mn(R)A, B \in M_n(\mathbb{R})A,B∈Mn​(R) 是两个 n×nn \times nn×n 的实数矩阵。假设它们在复数域上是相似的，即存在一个可逆的复矩阵 P∈Mn(C)P \in M_n(\mathbb{C})P∈Mn​(C)，使得 P−1AP=BP^{-1}A P = BP−1AP=B。
证明：它们在实数域上也必然相似（即存在一个可逆的实矩阵 Q∈Mn(R)Q \in M_n(\mathbb{R})Q∈Mn​(R)，使得 Q−1AQ=BQ^{-1}A Q = BQ−1AQ=B）。

已知 P−1AP=BP^{-1}A P = BP−1AP=B，可以改写为 AP=PBAP = PBAP=PB。因为 PPP 是复矩阵，我们可以将其拆分为实部和虚部：P=X+iYP = X + iYP=X+iY，其中 X,Y∈Mn(R)X, Y \in M_n(\mathbb{R})X,Y∈Mn​(R) 都是实矩阵。将拆分后的 PPP 代入等式：
A(X+iY)=(X+iY)BA(X + iY) = (X + iY)BA(X+iY)=(X+iY)B
展开得到：
AX+iAY=XB+iYBAX + iAY = XB + iYBAX+iAY=XB+iYB
因为 A,B,X,YA, B, X, YA,B,X,Y 都是实矩阵，复数相等意味着实部与实部相等，虚部与虚部相等。因此我们得到两个纯实数域的等式：
AX=XBAX = XBAX=XB
AY=YBAY = YBAY=YB
考虑利用 XXX 和 YYY 构造出一个可逆的实矩阵 QQQ。由于 P=X+iYP = X + iYP=X+iY 可逆，已知 det⁡(X+iY)≠0\det(X + iY) \neq 0det(X+iY)=0。
考虑构造一个关于实变量 ttt 的实系数多项式函数：f(t)=det⁡(X+tY)f(t) = \det(X + tY)f(t)=det(X+tY) 因为 X,YX, YX,Y 是实矩阵，显然 f(t)f(t)f(t) 是一个关于 ttt 的 nnn 次实系数多项式。

根据条件，当 t=it = it=i 时，f(i)=det⁡(X+iY)=det⁡(P)≠0f(i) = \det(X + iY) = \det(P) \neq 0f(i)=det(X+iY)=det(P)=0。既然 f(i)≠0f(i) \neq 0f(i)=0，说明 f(t)f(t)f(t) 绝不是一个恒等于零的多项式。既然 f(t)f(t)f(t) 不是零多项式，根据代数基本定理，它最多只有 nnn 个根。所以在实数中，必然存在某个实数 t0t_0t0​，使得 f(t0)≠0f(t_0) \neq 0f(t0​)=0。令 Q=X+t0YQ = X + t_0 YQ=X+t0​Y。显然 QQQ 是一个实矩阵，且因为 det⁡(Q)=f(t0)≠0\det(Q) = f(t_0) \neq 0det(Q)=f(t0​)=0，所以 QQQ 是可逆的。

验证一下：
AQ=A(X+t0Y)=AX+t0AYAQ = A(X + t_0 Y) = AX + t_0 AYAQ=A(X+t0​Y)=AX+t0​AY
根据前面的结论，替换掉 AXAXAX 和 AYAYAY：
AQ=XB+t0YB=(X+t0Y)B=QBAQ = XB + t_0 YB = (X + t_0 Y)B = QBAQ=XB+t0​YB=(X+t0​Y)B=QB
因为 QQQ 可逆，左乘 Q−1Q^{-1}Q−1 得到 Q−1AQ=BQ^{-1}AQ = BQ−1AQ=B。

请构造两个 2×22 \times 22×2 的实矩阵 AAA 和 BBB，使得它们满足 eA+B=eAeBe^{A+B} = e^A e^BeA+B=eAeB，但是 AB≠BAAB \neq BAAB=BA。

我们知道矩阵指数是有周期性的，如果一个实矩阵的特征值是 ±2kπi\pm 2k\pi i±2kπi，它的指数矩阵就会变成单位阵 III。

考虑构造矩阵，使得 eA=Ie^A = IeA=I，eB=Ie^B = IeB=I，且 eA+B=Ie^{A+B} = IeA+B=I。这样 eAeB=I⋅I=I=eA+Be^A e^B = I \cdot I = I = e^{A+B}eAeB=I⋅I=I=eA+B 自然成立。

构造 AAA：

令 A=2π(01−10)A = 2\pi \begin{pmatrix} 0 & 1 \\ -1 & 0 \end{pmatrix}A=2π(0−1​10​)。
容易计算 tr(A)=0\text{tr}(A) = 0tr(A)=0 且 det⁡(A)=4π2\det(A) = 4\pi^2det(A)=4π2。它的特征方程是 λ2+4π2=0\lambda^2 + 4\pi^2 = 0λ2+4π2=0，特征值为 ±2πi\pm 2\pi i±2πi。
所以 AAA 相似于对角阵 diag(2πi,−2πi)\text{diag}(2\pi i, -2\pi i)diag(2πi,−2πi)，因此 eA=Ie^A = IeA=I。

构造 BBB：

我们希望 BBB 也有特征值 ±2πi\pm 2\pi i±2πi，即要求 tr(B)=0,det⁡(B)=4π2\text{tr}(B) = 0, \det(B) = 4\pi^2tr(B)=0,det(B)=4π2。
同时，我们希望 A+BA+BA+B 的特征值是 ±6πi\pm 6\pi i±6πi（也是 2πi2\pi i2πi 的偶数倍，保证指数为 III）。即要求 tr(A+B)=0,det⁡(A+B)=36π2\text{tr}(A+B) = 0, \det(A+B) = 36\pi^2tr(A+B)=0,det(A+B)=36π2。
我们大胆乱搞，令 B=2π(56−1−5)B = 2\pi \begin{pmatrix} \sqrt{5} & 6 \\ -1 & -\sqrt{5} \end{pmatrix}B=2π(5​−1​6−5​​)。
验证一下： tr(B)=0\text{tr}(B) = 0tr(B)=0，det⁡(B)=(2π)2(−5⋅5−(−6))=4π2(−5+6)=4π2\det(B) = (2\pi)^2 (-\sqrt{5}\cdot\sqrt{5} - (-6)) = 4\pi^2(-5+6) = 4\pi^2det(B)=(2π)2(−5​⋅5​−(−6))=4π2(−5+6)=4π2。
所以 BBB 的特征值为 ±2πi\pm 2\pi i±2πi，因此 eB=Ie^B = IeB=I。

A+B=2π(57−2−5)A+B = 2\pi \begin{pmatrix} \sqrt{5} & 7 \\ -2 & -\sqrt{5} \end{pmatrix}A+B=2π(5​−2​7−5​​) tr(A+B)=0\text{tr}(A+B) = 0tr(A+B)=0 det⁡(A+B)=(2π)2(−5⋅5−(−14))=4π2(−5+14)=36π2=(6π)2\det(A+B) = (2\pi)^2 (-\sqrt{5}\cdot\sqrt{5} - (-14)) = 4\pi^2(-5+14) = 36\pi^2 = (6\pi)^2det(A+B)=(2π)2(−5​⋅5​−(−14))=4π2(−5+14)=36π2=(6π)2

这说明 A+BA+BA+B 的特征值为 ±6πi\pm 6\pi i±6πi。因此 eA+B=Ie^{A+B} = IeA+B=I。

至此我们有 eAeB=I⋅I=I=eA+Be^A e^B = I \cdot I = I = e^{A+B}eAeB=I⋅I=I=eA+B，验证它们是否可交换：
AB=4π2(01−10)(56−1−5)=4π2(−1−5−5−6)AB = 4\pi^2 \begin{pmatrix} 0 & 1 \\ -1 & 0 \end{pmatrix} \begin{pmatrix} \sqrt{5} & 6 \\ -1 & -\sqrt{5} \end{pmatrix} = 4\pi^2 \begin{pmatrix} -1 & -\sqrt{5} \\ -\sqrt{5} & -6 \end{pmatrix}AB=4π2(0−1​10​)(5​−1​6−5​​)=4π2(−1−5​​−5​−6​)
BA=4π2(56−1−5)(01−10)=4π2(−655−1)BA = 4\pi^2 \begin{pmatrix} \sqrt{5} & 6 \\ -1 & -\sqrt{5} \end{pmatrix} \begin{pmatrix} 0 & 1 \\ -1 & 0 \end{pmatrix} = 4\pi^2 \begin{pmatrix} -6 & \sqrt{5} \\ \sqrt{5} & -1 \end{pmatrix}BA=4π2(5​−1​6−5​​)(0−1​10​)=4π2(−65​​5​−1​)
显然 AB≠BAAB \neq BAAB=BA。

考虑矩阵方程： det⁡(A+B)+det⁡(A−B)=2det⁡(A)+2det⁡(B)\det(A+B) + \det(A-B) = 2\det(A) + 2\det(B)det(A+B)+det(A−B)=2det(A)+2det(B)。

  • 证明：对于任意的 2×22 \times 22×2 矩阵 A,BA, BA,B，该等式恒成立。
  • 证明：对于 3×33 \times 33×3 矩阵，该等式不一定成立，并给出反例。

n=2n=2n=2

设 A=[a1,a2],B=[b1,b2]A = [a_1, a_2], B = [b_1, b_2]A=[a1​,a2​],B=[b1​,b2​]，其中 ai,bia_i, b_iai​,bi​ 是二维列向量。
利用行列式关于列向量的多重线性展开：

det⁡(A+B)=det⁡([a1+b1,a2+b2])\det(A+B) = \det([a_1+b_1, a_2+b_2])det(A+B)=det([a1​+b1​,a2​+b2​]) =det⁡(a1,a2)+det⁡(a1,b2)+det⁡(b1,a2)+det⁡(b1,b2)= \det(a_1, a_2) + \det(a_1, b_2) + \det(b_1, a_2) + \det(b_1, b_2)=det(a1​,a2​)+det(a1​,b2​)+det(b1​,a2​)+det(b1​,b2​)

注意到 det⁡(a1,a2)=det⁡(A)\det(a_1, a_2) = \det(A)det(a1​,a2​)=det(A)，det⁡(b1,b2)=det⁡(B)\det(b_1, b_2) = \det(B)det(b1​,b2​)=det(B)，所以：

det⁡(A+B)=det⁡(A)+det⁡(B)+det⁡(a1,b2)+det⁡(b1,a2)\det(A+B) = \det(A) + \det(B) + \det(a_1, b_2) + \det(b_1, a_2)det(A+B)=det(A)+det(B)+det(a1​,b2​)+det(b1​,a2​)

同理展开 det⁡(A−B)\det(A-B)det(A−B)：

det⁡(A−B)=det⁡([a1−b1,a2−b2])\det(A-B) = \det([a_1-b_1, a_2-b_2])det(A−B)=det([a1​−b1​,a2​−b2​]) =det⁡(a1,a2)−det⁡(a1,b2)−det⁡(b1,a2)+det⁡(−b1,−b2)= \det(a_1, a_2) - \det(a_1, b_2) - \det(b_1, a_2) + \det(-b_1, -b_2)=det(a1​,a2​)−det(a1​,b2​)−det(b1​,a2​)+det(−b1​,−b2​)

注意最后一项中可以提取出两个负号：det⁡(−b1,−b2)=(−1)2det⁡(b1,b2)=det⁡(B)\det(-b_1, -b_2) = (-1)^2 \det(b_1, b_2) = \det(B)det(−b1​,−b2​)=(−1)2det(b1​,b2​)=det(B)。所以：

det⁡(A−B)=det⁡(A)+det⁡(B)−det⁡(a1,b2)−det⁡(b1,a2)\det(A-B) = \det(A) + \det(B) - \det(a_1, b_2) - \det(b_1, a_2)det(A−B)=det(A)+det(B)−det(a1​,b2​)−det(b1​,a2​)

两式相加，交叉项刚好抵消。

det⁡(A+B)+det⁡(A−B)=2det⁡(A)+2det⁡(B)\det(A+B) + \det(A-B) = 2\det(A) + 2\det(B)det(A+B)+det(A−B)=2det(A)+2det(B)

因此等式在 2×22 \times 22×2 情况下成立。
